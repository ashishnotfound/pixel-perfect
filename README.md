# Reyo Studio Portfolio & Dynamic QR Landing Page

Modern, high-performing portfolio website and QR landing page for **Reyo Studio**.

## Features

- **Dynamic Hero & Document Title**: Automatically adjusts messaging depending on URL query parameter `?from=qr`.
- **Portfolio Showcase**: Highlights real business websites and applications built by Reyo Studio.
- **Responsive & Accessible Design**: Crafted using React, TypeScript, and Tailwind CSS.
- **Fast Performance**: Optimized layouts, images, and minimal client overhead.

## Development

To run the development server locally:

```sh
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
To preview the QR visitor experience, append `?from=qr` to the URL: [http://localhost:5173/?from=qr](http://localhost:5173/?from=qr).

## Tech Stack

- Vite / React 19
- TanStack Router
- TypeScript
- Tailwind CSS
- Lucide React
